# Garuda Quest 🦅

**Garuda Quest** adalah gim edukasi interaktif berbasis logika lateral (berkonsep *teka-teki ala Quiz Parampa*) yang dirancang khusus sebagai media pembelajaran materi Wawasan Kebangsaan. Proyek ini dikembangkan sebagai bagian dari tugas akhir (Skripsi) di Program Studi Teknik Informatika, Universitas Islam Balitar (UNISBA) Blitar.

Gim ini bertujuan untuk mendobrak metode pembelajaran konvensional dengan menghadirkan pengalaman "belajar sambil bermain" bagi siswa SMP, khususnya target implementasi di SMP Negeri 8 Kota Blitar.

---

## 🎮 Fitur Utama
- **Mekanika Logika Lateral:** Soal-soal yang disajikan tidak hanya menguji hafalan, tetapi juga logika *out-of-the-box* pemain.
- **Sistem Nyawa (*Lives-based*):** Menggantikan batasan waktu (*timer*) dengan sistem nyawa (maksimal 3) untuk mengurangi tekanan pada siswa saat berpikir.
- **Sistem Pangkat/Tier:** Fitur apresiasi pemain menggunakan jenjang kepangkatan bertema nasionalisme (seperti "Tunas Bangsa").
- **Hidden Objects & Multiple Win-States:** Elemen rahasia di dalam level yang memicu rasa ingin tahu dan eksplorasi pemain.

---

## 🛠️ Tech Stack & Tools
Proyek ini dibangun dan didesain menggunakan berbagai perangkat lunak berikut:
- **Construct 2:** *Game Engine* utama untuk merancang arsitektur logika, *Event Sheet*, dan *gameplay*.
- **Canva & Affinity:** Pembuatan aset visual, antarmuka (*User Interface*), dan grafis *pixel art*.
- **Audacity:** Perekaman, pemotongan, dan pengeditan efek suara (*SFX*) serta musik latar.
- **Visual Studio Code (VS Code):** *Code editor* untuk manajemen berkas dan sinkronisasi *version control* ke GitHub.
